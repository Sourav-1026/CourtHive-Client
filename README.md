# 🏟️ CourtHive

**CourtHive** is a full-stack web application that allows users to browse, list, and book sports courts — including futsal, badminton, and cricket. Whether you're looking to book a court for a quick match or want to earn by listing your own facility, CourtHive makes it seamless.

---

## 🚀 Features

### 1. 🔐 Authentication (Email & Google)

Users can sign up and log in using their email and password, or instantly access the platform via **Google OAuth** — making onboarding fast and frictionless.

### 2. 🏟️ Court Listings

Browse a wide variety of sports courts listed by other users. Each court displays key details like sport type, capacity, hourly rate, and available amenities to help you find the perfect place to play.

### 3. ➕ Add Your Own Court

Have a facility to share? Users can list their own courts with details such as name, description, images, rate, sport type, capacity, and amenities — and start earning from bookings.

### 4. 📅 Court Booking System

Book any available court by selecting your desired time slot. The platform handles scheduling under the hood to ensure a smooth booking experience.

### 5. ⚠️ Booking Conflict Detection

A smart conflict detection system on the backend prevents double bookings. If a time slot is already taken, users are notified immediately — keeping all reservations clean and reliable.

### 6. ❌ Booking Cancellation

Plans changed? Users can cancel their existing bookings with ease, freeing up the court for others to reserve.
